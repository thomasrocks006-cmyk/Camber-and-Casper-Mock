2. Core Components
2.1 Apollo.io Integration
Purpose: Contact discovery with verified emails and phones

API Endpoint: https://api.apollo.io/v1/mixed_people/search

Search Criteria:

{
    "person_titles": [
        "Head of Sales", "Sales Director", "VP Sales",
        "CEO", "Founder"
    ],
    "organization_locations": ["Australia"],
    "organization_num_employees_ranges": ["10,500"],  # SMB sweet spot
    "person_seniorities": ["director", "vp", "c_suite", "owner"],
    "contact_email_status": ["verified"]  # ✅ Only verified emails
}
Output: List of contacts with:

Verified work email
Direct phone numbers (when available)
LinkedIn profile
Company details
Job title and seniority
Status: ✅ Fully functional (API key configured)

2.2 SEEK Job Scraping
Purpose: Detect hiring signals (companies growing sales teams)

Trigger Type: SEEK_JOB_POSTING

How it Works:

Search SEEK.com.au for company name + "sales OR BDM OR business development"
Parse job cards (BeautifulSoup)
If company posted sales/BDM role → HOT SIGNAL (hiring = growth)
Signal Confidence: High

Example:

Company: Acme Corp
SEEK Result: "Business Development Manager - Sydney"
Trigger: ✅ SEEK_JOB_POSTING
Description: "Posted job for 'Business Development Manager' on SEEK"
Status: ✅ Fully functional (scraping enabled)

2.3 ABN Lookup
Purpose: Business registration monitoring (new businesses)

Trigger Type: ABN_REGISTRATION

API: ABN Lookup API (Australian Business Register)

How it Works:

Check company ABN registration date
If registered < 12 months → New business (needs systems)
Extract industry, state, status
Status: ⚠️ Stub implementation (needs ABN API key)

2.4 Decision-Maker Scorer
Module: agents/scout/decision_maker_scorer.py (611 lines)

Purpose: Score authority level of contact (0-100)

Scoring Factors:

Department Type: Sales (100), Marketing (90), Operations (70), IT (60), Other (40)
Decision Level: C-Suite (100), Director (80), Manager (60), Coordinator (40)
Seniority Indicators: "Head of", "VP", "Chief", "Director"
Budget Authority: Inferred from title
Influencer vs. Economic Buyer: Classified
Score Ranges:

90-100: Economic buyer (C-suite, direct budget)
70-89: High influencer (Director, VP)
50-69: Moderate influencer (Manager)
< 50: Low influencer (Coordinator, Specialist)
Output:

ContactScore(
    contact_id="apollo_123",
    decision_level=DecisionLevel.DIRECTOR,
    department=DepartmentType.SALES,
    authority_score=85,
    budget_authority=True,
    primary_decision_maker=True,
    confidence=0.87
)
Status: ✅ Fully functional (imported from old production module)
